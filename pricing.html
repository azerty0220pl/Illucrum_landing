What’s included in each tier

Basic tier

Included in all three audit types

  • Technical SEO: page speed, Core Web Vitals, mobile, HTTPS, sitemap, robots.txt, canonical tags, schema
  • On-page SEO: titles, meta descriptions, headings, alt text, content quality, internal linking, URLs
  • Keyword strategy: primary targeting, search intent alignment
  • UX & conversion: CTA clarity, navigation, trust signals
  • Vertical-specific foundations
  • Scored report with traffic-light ratings
  • Top 5–7 priority action plan
  • 30-minute debrief session

Full tier

Everything in Basic, plus:

  • Backlink profile: referring domains, anchor text, toxic link detection
  • Competitor analysis: top 3 competitors, content gaps, SERP features
  • Extended keyword mapping (branded vs non-branded, FAQ/long-tail)
  • Full specialist coverage per audit type
  • Comprehensive prioritised roadmap
  • 45-minute debrief session
